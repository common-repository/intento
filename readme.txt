Ajout automatiquement notre tag de tracking dans le head de votre site.

== Description ==

En plus de l'ajout automatique du tag sur toutes les pages de votre site, ce plugin exclut automatique le trafic des utilisateurs authentifiés sur Wordpress.

L'ajout du tag intento› permet la collecte et l'envoi de données analytiques et comportementales à nos serveurs en Europe via notre api http://t.i-t-t.co.

Notre politique de collecte de données personnelles : https://www.intento.io/donnees-personnelles/

== Installation ==

1. Uploadez les fichiers dans le dossier `/wp-content/plugins/intento` or utilisez le store de wordpress directement
2. Activez le plugin dans la section "extensions" de votre interface d'administration
3. Le tag est installé

Avant toute installation, rapprochez vous de notre équipe, qui vous ouvrira un espace sur notre plateforme intento›.
